Budgety for Udemy course: The Complete JavaScript Course 2020: Build Real Projects!

In this project we learned:
* How to think and plan to build the project with coding.
* How to structure code with modules.
* Private and public data, encapsulation, separation of concerns.
* How to set up event listeners for keypress events.
* How to use event object.
* How and why make an initialization function.
